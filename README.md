# Business Card PCB
This repo includes the KiCad files for a PCB I designed that will function as a business card. The hardware on top of the PCB will allow the use
to play a simple "Simon Says" style of game, where LEDs will display a pattern of lights that the user will have to replicate by pressing three tactile buttons.


## Components
- ATtiny85
- 4 tactile buttons
- Piezo buzzer
- 3 Red LEDs
- 1 Yellow LED
- 1 Green LED
